======================================================================
PROJETO - E-COMMERCE DE AUTOPEÇAS (CARROS E MOTOS)
======================================================================

OBJETIVO

Desenvolver uma plataforma web para venda de autopeças utilizando apenas:

- HTML
- CSS
- JavaScript
- Python (FastAPI)
- SQLite

O sistema permitirá que o usuário:

• visualizar produtos em promoção;
• navegar por categorias;
• pesquisar produtos pelo nome;
• pesquisar utilizando filtros;
• visualizar detalhes dos produtos.

======================================================================
COMO O SISTEMA FUNCIONA
======================================================================

                USUÁRIO
                    │
                    ▼
        HTML + CSS + JavaScript
                    │
            Requisições HTTP
                    │
                    ▼
            Backend (Python)
                    │
                    ▼
             Banco de Dados
                 SQLite

IMPORTANTE

O Frontend NÃO acessa o banco de dados.

Todo acesso ao banco será feito SOMENTE pelo Backend.

======================================================================
PADRÃO DE VISUAL
======================================================================

Antes de criar qualquer página devemos seguir o mesmo padrão.

Todos devem utilizar:

- mesmas cores
- mesma fonte
- mesmo estilo dos botões
- mesmo estilo dos cards
- mesmo tamanho dos títulos

Isso evita que cada página fique diferente.

======================================================================
COMO TRABALHAR NO GITHUB
======================================================================

NUNCA alterar diretamente a branch main.

Cada integrante trabalhará em sua própria branch.

Exemplo:

feature-produtos => João

feature-backend => Josué

feature-database => Luis

Quando terminar uma funcionalidade, solicitar a integração (merge).
